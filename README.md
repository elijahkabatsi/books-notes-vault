BOOKS AND NOTES VAULT

OVERVIEW

Welcome to Imari Books!
Imari is a kinyarwanda word for vault. It is a metal case with a lock students ususally take to boarding school, and inside mine was some of my favourite books.
Here, you can explore some of the most famous books, search for your favorites, and stay updated on stats about reading trends. Our mission is to make literature fun and accessible for everyone.
Enjoy Reading!

FEATURES

Books: Options of books to read.
Search: You can search for a book in the website.
Notes: You can take notes, save them, or delete them.
About: You can learn about the website and its creator with visual aid.

Technologies

Frontend: HTML, CSS, JavaScript
Styling: Custom CSS with modern layout and design
Functionality: JavaScript for search, flashcards, and state management
